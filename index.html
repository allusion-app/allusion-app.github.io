Get Allusion

A great image organizer at the core

Allusion only needs to know where to look, and it will automatically load any content it finds at the specified location. This makes it so that saving a picture to your set folder will automatically add it to your library.

The Allusion browser extension for Google Chrome makes it easy to save images directly to your visual library. You can even add tags without ever having to leave your browser.

Allusion is meant to be an interface between you and your data. You can save it to any folder you want, where it will stay untouched.

An innovative approach to tagging

Anyone who has used tags before knows that at some point the list gets so long that one forgets what tags exist in the first place. This problem is entirely solved by managing tags in a hierarchy instead.

With one click, you can easily bring up a list of all images that have no tags assigned to them yet. This makes it so you never forget to tag an image again.

Custom colors on tags can be used to make them visually stand out. This is especially useful when browsing through a long list of images.

A powerful search

The easiest way of finding an image is by typing the name of a tag into the search bar at the top. The autocomplete feature lets you find tags even if you don't remember them clearly.

Whether you are looking at the import locations or your tag hierarchy, a single click will let you filter for any content that belongs to the entry.

Should the basic search not fulfill your needs, the advanced interface lets you combine queries to find the most hidden image of your library.
